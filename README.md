# Customer_behavior_end_to_end_analysis

## Overview 
This project demostrates an end-to-end data analytics workflow for analyzing customer behavior. It covers data
loading, cleaning, exploratory data analysis(EDA), SQL-based querying, interactive dashboard creation and business 
reporting. The goal is to extract meaningful insights that can support data-driven decision making.

## Dataset
The dataset contains customer-related information such as demographics, purchasing behavior, transactions and 
other relevant attributes.

## Key Fatures:
- Customer ID
- Age
- Gender
- Location
- Purchase amount
- Product category
- Customer rating
- Payment method
